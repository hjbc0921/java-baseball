# 우아한테크코스 1주차 미션 : 숫자 야구 게임

## 기능 요구사항

1. 컴퓨터가 random으로 세 자리 수를 결정한다.
2. user의 세 자리 input을 받는다.
3. input validity를 체크하고, 부적합하면 다시 2번으로 돌아간다.
4. 정답과 비교한다.
5. 알맞은 hint string (strike/ball)을 출력한다.
6. 정답이 아니면 2-4을 반복한다.
7. 정답이면 다시 실행 (1번으로 돌아감) 또는 종료 여부를 받는다.
